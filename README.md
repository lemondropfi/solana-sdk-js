# roundup-sdk

A TypeScript SDK for your project.

## Getting Started

### Installation

```
npm install roundup-sdk
```

### Usage

```
import { hello } from 'roundup-sdk';

console.log(hello('World'));
```

## Development

### Build

```
npm run build
```

### Clean

```
npm run clean
```

## Publishing

To publish to npm:

1. Update the version in `package.json`.
2. Run:
   ```
   npm publish
   ```
